Overview

This is a guide to mdBook for users of Windows 10. mdBook is a command line tool to create books. Books are created from source files that are written in Markdown. Each book is a set of HTML pages that can be uploaded as a static website. The contents of the book are displayed as a sidebar menu.

In keeping with the lightweight nature of Markdown, websites produced with mdBook are simple and uncluttered. There is a range of standard themes, each of which can be fully customised. A search box is included, with highlighting of search results.

mdBook is written in Rust. Most of the available online resources are aimed at users running Node.js and npm. However, mdBook is also available as a single compiled file, and is ideal for Windows users who are seeking to generate a static website.

mdBook is free and open source, with source code on GitHub. Issues and feature requests can be posted on the issue tracker.

Requirements

mdBook is a command line tool, so the user requires a basic understanding of running commands from the Windows command prompt. It can also be used through PowerShell.

mdBook makes use of source files written in Markdown, so the user needs to learn some simple syntax. As Markdown is a text format, it can be edited in any text editor. However, use of a dedicated Markdown editor simplifies the task. A recommended choice is the open source editor Zettlr.

Books can be viewed offline in a modern browser such as Google Chrome or Microsoft Edge. In order to view books online, the user needs to be able to upload files to some webspace. This is commonly done with a FTP client such as Filezilla.

Setup

Setup of mdBook in Windows starts with download of the latest Windows release from the page at:

https://github.com/rust-lang/mdBook/releases

At the time of writing the current release is v0.3.7, which is contained in the ZIP file:

mdbook-v0.3.7-x86_64-pc-windows-msvc.zip

This archive contains just one file - the compiled version of mdBook:

mdbook.exe

Extract the file to a folder to hold the books that you will create with mdBook. Let us call the folder mdBooks. Now open a command prompt window and navigate to the new folder. To initialise a new book called Book1, create a sub-folder called Book1 and run the command:

mdbook init Book1 --theme

This will create the following structure in folder Book1:

├───book
│   ├───css
│   └───FontAwesome
│       ├───css
│       └───fonts
└───src
    └───theme
        └───css

With the current release, it is necessary to manually move the theme folder up a level prior to building the book. After the move the structure in folder Book1 becomes:

├───book
│   ├───css
│   └───FontAwesome
│       ├───css
│       └───fonts
├───src
└───theme
    └───css

We are now ready to format the book.
